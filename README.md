Memory Match: Minimalist Edition
A sleek and responsive memory matching game built with vanilla JavaScript and CSS. This project focuses on smooth state management and a refined minimalist pink aesthetic with a retro 90s flair.

Project Overview
The game challenges players to find matching pairs of icons within a grid. It features a custom built card flipping mechanic and real time game state tracking to ensure a seamless user experience.

Core Mechanics
Grid Logic: Dynamic card generation using JavaScript arrays.

Shuffle System: Randomized card placement using array shuffling on every initialization.

State Management: Robust logic to handle first and second picks while locking the board during animations.

3D Animations: Implemented using CSS perspective and backface visibility for a realistic flip effect.

Design Features
90s Splash Screen: A retro inspired "Press Start" entry point with blinking animations.

Minimalist Aesthetic: A sophisticated flat pink color palette without the use of gradients.

Clean UI: Focused on high contrast typography and clear visual feedback for matches.

Responsive Layout: The grid adapts to different screen sizes for a consistent experience across devices.

Stretch Features
Live Timer: Monitors the duration of the gameplay session.

Score Tracking: Displays the number of successful matches in real time.

Restart Function: A dedicated reset to clear the board and timer without refreshing the browser.

Technical Stack
HTML5

CSS3 (Flexbox, Grid, 3D Transforms)

JavaScript (DOM Manipulation, ES6 Syntax)
